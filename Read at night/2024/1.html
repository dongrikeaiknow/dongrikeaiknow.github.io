✨ 什么是糖酵解和柠檬酸循环？

糖酵解和柠檬酸循环是细胞呼吸的两个关键代谢途径！它们将葡萄糖中的化学能转化为细胞可用的能量形式——ATP！

糖酵解 (Glycolysis)

在细胞质中进行！将1分子葡萄糖分解为2分子丙酮酸，产生2ATP和2NADH！

总反应式： C6H12O6 + 2NAD+ + 2ADP + 2Pi → 2CH3COCOO- + 2NADH + 2ATP + 2H2O + 2H+

柠檬酸循环 (Citric Acid Cycle)

在线粒体基质中进行！将丙酮酸彻底氧化，产生大量NADH和FADH2！

关键产物： 3NADH + 1FADH2 + 1GTP/ATP (每轮循环)

🔬 糖酵解详细步骤 (10步反应)

1. 葡萄糖磷酸化

己糖激酶催化葡萄糖 → 葡萄糖-6-磷酸 (G6P)

消耗1ATP！这是糖酵解的"活化能投资"阶段！

2. 异构化反应

G6P → 果糖-6-磷酸 (F6P)

磷酸葡萄糖异构酶催化醛糖→酮糖的转变！

3. 二次磷酸化

F6P → 果糖-1,6-二磷酸 (F1,6BP)

磷酸果糖激酶-1(PFK-1)催化，再消耗1ATP！

4. 裂解反应

F1,6BP → 二羟丙酮磷酸 (DHAP) + 甘油醛-3-磷酸 (G3P)

醛缩酶催化六碳糖裂解为两个三碳糖！

5. 异构化反应

DHAP ⇌ G3P (由磷酸丙糖异构酶催化)

最终得到2分子G3P进入后续反应！

6. 氧化磷酸化

G3P → 1,3-二磷酸甘油酸 (1,3BPG)

产生NADH！这是能量捕获的关键步骤！

7. ATP生成

1,3BPG → 3-磷酸甘油酸 (3PG)

磷酸甘油酸激酶催化，生成ATP！

8. 分子重排

3PG → 2-磷酸甘油酸 (2PG)

磷酸甘油酸变位酶催化磷酸基团转移！

9. 脱水反应

2PG → 磷酸烯醇式丙酮酸 (PEP)

烯醇化酶催化，形成高能磷酸键！

10. ATP生成

PEP → 丙酮酸 (Pyruvate)

丙酮酸激酶催化，再生成ATP！

💡 糖酵解总结

1分子葡萄糖 → 2分子丙酮酸 + 2ATP (净得) + 2NADH！

后续丙酮酸可进入：

  • 有氧条件：进入线粒体→柠檬酸循环
  • 无氧条件：发酵→乳酸或乙醇

🌀 柠檬酸循环详细步骤 (8步反应)

1. 丙酮酸脱羧

丙酮酸 → 乙酰CoA + CO2

丙酮酸脱氢酶复合体催化，产生1NADH！

2. 柠檬酸合成

乙酰CoA + 草酰乙酸 → 柠檬酸

柠檬酸合酶催化，释放CoA-SH！

3. 异构化反应

柠檬酸 → 异柠檬酸

顺乌头酸酶催化，为后续脱氢做准备！

4. 第一次氧化脱羧

异柠檬酸 → α-酮戊二酸 + CO2

异柠檬酸脱氢酶催化，产生1NADH！

5. 第二次氧化脱羧

α-酮戊二酸 → 琥珀酰CoA + CO2

α-酮戊二酸脱氢酶复合体催化，产生1NADH！

6. 底物水平磷酸化

琥珀酰CoA → 琥珀酸

琥珀酰CoA合成酶催化，产生1GTP (相当于ATP)！

7. 第三次氧化

琥珀酸 → 延胡索酸

琥珀酸脱氢酶催化，产生1FADH2！

8. 再生草酰乙酸

延胡索酸 → 苹果酸 → 草酰乙酸

苹果酸脱氢酶催化，产生1NADH，完成循环！

💡 柠檬酸循环总结

1分子乙酰CoA → 3NADH + 1FADH2 + 1GTP + 2CO2！

加上丙酮酸→乙酰CoA阶段：

  • 1分子葡萄糖 → 6NADH + 2FADH2 + 2GTP + 4CO2
  • NADH和FADH2进入电子传递链产生更多ATP！

⚡ 能量计算 - 葡萄糖完全氧化

糖酵解阶段

  • 2ATP (净得)
  • 2NADH → 5ATP*

总计：7ATP

丙酮酸氧化

  • 2NADH → 5ATP*

总计：5ATP

柠檬酸循环

  • 6NADH → 15ATP*
  • 2FADH2 → 3ATP*
  • 2GTP → 2ATP

总计：20ATP

📊 总能量产出

1葡萄糖 → ~32ATP*

*基于1NADH≈2.5ATP，1FADH2≈1.5ATP的化学渗透理论计算

🔍 效率分析

葡萄糖完全燃烧释放~2870kJ/mol，32ATP储存~32×30.5=976kJ

能量转化效率约34%，远高于内燃机(20-30%)！

🎛️ 关键酶与代谢调控

糖酵解关键调控点

己糖激酶

受G6P反馈抑制！防止产物积累！

磷酸果糖激酶-1 (PFK-1)

最重要调控点！受ATP/AMP、柠檬酸、pH等调节！

ATP↑抑制，AMP↑激活，柠檬酸↑抑制

丙酮酸激酶

受F1,6BP激活，ATP/丙氨酸抑制！

柠檬酸循环关键调控点

丙酮酸脱氢酶复合体

受产物抑制 (NADH、乙酰CoA) 和磷酸化/去磷酸化调节！

异柠檬酸脱氢酶

受NADH和ATP抑制，ADP和Ca2+激活！

α-酮戊二酸脱氢酶复合体

受产物抑制 (NADH、琥珀酰CoA)！
