氨基酸的脱氨基作用 ✨

氧化脱氨基作用

氨基酸在酶催化下氧化生成α-酮酸，消耗氧并产生氨！这是氨基酸分解代谢的重要方式之一！

R-CH(NH₃⁺)-COO⁻ + H₂O + O₂ → R-CO-COO⁻ + NH₄⁺ + H₂O₂

关键酶：L-氨基酸氧化酶、D-氨基酸氧化酶和L-谷氨酸脱氢酶！

转氨基作用

氨基酸的α-氨基转移到α-酮酸上，生成新的氨基酸和新的α-酮酸！这是氨基酸代谢中最常见的反应！

氨基酸₁ + α-酮酸₂ ⇌ α-酮酸₁ + 氨基酸₂

辅酶：磷酸吡哆醛（维生素B6的活性形式）！

联合脱氨基作用

转氨基作用与谷氨酸氧化脱氨基作用联合进行！这是体内氨基酸脱氨基的主要方式！

1. 氨基酸 + α-酮戊二酸 → α-酮酸 + 谷氨酸
2. 谷氨酸 + NAD⁺ + H₂O → α-酮戊二酸 + NADH + NH₄⁺

嘌呤核苷酸循环是另一种联合脱氨基方式！

脱酰胺基作用

天冬酰胺和谷氨酰胺的酰胺基由相应的酰胺酶加水脱去氨基！这是特定氨基酸的特殊代谢途径！

天冬酰胺 + H₂O → 天冬氨酸 + NH₄⁺
谷氨酰胺 + H₂O → 谷氨酸 + NH₄⁺

酰胺酶在脑、肾等组织中活性较高！

氨的代谢去路与鸟氨酸循环 🌀

氨的排泄方式

不同生物对氨的排泄方式不同，反映了进化过程中的环境适应！

  • 排氨动物：水生动物，直接排氨（毒性大但水稀释）
  • 排尿动物：两栖类，将氨转化为尿素（中等毒性，中等水溶性）
  • 排尿酸动物：鸟类和爬行类，将氨转化为尿酸（毒性低，几乎不溶于水）

人类和哺乳动物主要通过尿素循环将氨转化为尿素排出！

氨的转运方式

氨在体内需要安全转运，主要有两种方式！

1. 以谷氨酰胺形式转运：

谷氨酸 + NH₄⁺ + ATP → 谷氨酰胺 + ADP + Pi

2. 葡萄糖-丙氨酸循环：

肌肉中：丙酮酸 + 氨基酸 → 丙氨酸
肝脏中：丙氨酸 → 丙酮酸 + NH₄⁺（进入尿素循环）

谷氨酰胺是中性无毒物质，容易通过细胞膜！

鸟氨酸循环（尿素循环）

鸟氨酸循环是哺乳动物将有毒的氨转化为尿素的主要代谢途径！1932年由Hans Krebs发现！

循环过程：

  1. 氨甲酰磷酸合成：NH₃ + CO₂ + 2ATP → 氨甲酰磷酸
  2. 瓜氨酸合成：鸟氨酸 + 氨甲酰磷酸 → 瓜氨酸
  3. 精氨琥珀酸合成：瓜氨酸 + 天冬氨酸 + ATP → 精氨琥珀酸
  4. 精氨酸合成：精氨琥珀酸 → 精氨酸 + 延胡索酸
  5. 尿素生成：精氨酸 → 鸟氨酸 + 尿素

每生成1分子尿素消耗4个高能磷酸键（3ATP）！

⚛️

尿素循环与三羧酸循环的联系

1. 延胡索酸可转化为草酰乙酸进入TCA循环

2. 草酰乙酸可转氨生成天冬氨酸重新进入尿素循环

3. 两者共同构成"Krebs bicycle"（尿素-柠檬酸双循环）

氨基酸的合成代谢 🌱

氨基酸合成途径的六大类型

1. α-酮戊二酸衍生型

合成：谷氨酸、谷氨酰胺、脯氨酸、精氨酸

2. 草酰乙酸衍生型

合成：天冬氨酸、天冬酰胺、赖氨酸、甲硫氨酸、苏氨酸、异亮氨酸

3. 丙酮酸衍生型

合成：丙氨酸、缬氨酸、亮氨酸

4. 甘油酸-3-磷酸衍生型

合成：丝氨酸、甘氨酸、半胱氨酸

5. 赤藓糖-4-磷酸和PEP衍生型

合成：苯丙氨酸、酪氨酸、色氨酸

6. 组氨酸生物合成

由5'-磷酸核糖基焦磷酸(PRPP)开始合成

必需氨基酸

人体无法合成，必须从食物中获取的氨基酸！共8种：

苏氨酸 赖氨酸 甲硫氨酸 色氨酸 苯丙氨酸 缬氨酸 亮氨酸 异亮氨酸

婴幼儿还需组氨酸和精氨酸（半必需氨基酸）！

氨基酸的代谢去向

氨基酸在体内有多种代谢去向，构成复杂的代谢网络！

  • 合成蛋白质：主要去向，用于组织生长和修复
  • 合成其他含氮化合物：如嘌呤、嘧啶、肌酸等
  • 氧化供能：脱氨后α-酮酸进入三羧酸循环
  • 转变为糖或脂肪：生糖氨基酸和生酮氨基酸

氨基酸代谢要点总结 🎯

1. 脱氨基作用

包括氧化脱氨、转氨、联合脱氨等多种方式，是氨基酸分解的关键步骤！

2. 氨的代谢

氨通过尿素循环转化为尿素排出，与TCA循环紧密相连！

3. 合成代谢

氨基酸可从多种代谢中间物合成，分为六大类型！
