生物分子代谢可视化

糖类、脂类、蛋白质和核酸的代谢反应与相互转化

糖类代谢

糖酵解（EMP途径）

葡萄糖 → 丙酮酸 + ATP + NADH

关键酶：己糖激酶、磷酸果糖激酶、丙酮酸激酶

三羧酸循环（TCA循环）

乙酰CoA → CO₂ + ATP + NADH + FADH₂

关键酶：柠檬酸合酶、异柠檬酸脱氢酶、α-酮戊二酸脱氢酶

磷酸戊糖途径（PPP途径）

葡萄糖-6-磷酸 → 核糖-5-磷酸 + NADPH

为核酸合成提供原料和还原力！

」 糖代谢是细胞的能量主要来源，1分子葡萄糖完全氧化可产生约30-32分子ATP！

脂类代谢

脂肪酸β-氧化

脂肪酸 → 乙酰CoA + NADH + FADH₂

关键酶：肉碱脂酰转移酶I（限速酶）

脂肪酸合成

乙酰CoA + NADPH → 软脂酸

关键酶：乙酰CoA羧化酶（限速酶）

胆固醇代谢

乙酰CoA → 胆固醇

关键酶：HMG-CoA还原酶（限速酶）

」 1分子软脂酸完全氧化可产生106分子ATP，是糖类能量的2倍多！

蛋白质代谢

蛋白质降解

蛋白质 → 氨基酸

关键酶：组织蛋白酶、溶酶体蛋白酶

氨基酸脱氨

氨基酸 → α-酮酸 + NH₃

关键酶：转氨酶、谷氨酸脱氢酶

尿素循环

NH₃ + CO₂ → 尿素

关键酶：氨基甲酰磷酸合成酶I（限速酶）

」 必需氨基酸有8种：缬、异亮、亮、苯丙、蛋、色、苏、赖氨酸！

核酸代谢

嘌呤核苷酸合成

PRPP + 甘氨酸 + 谷氨酰胺等 → IMP

关键酶：PRPP合成酶、谷氨酰胺-PRPP-酰胺转移酶

嘧啶核苷酸合成

谷氨酰胺 + CO₂ + Asp → UMP

关键酶：氨基甲酰磷酸合成酶II

核苷酸降解

嘌呤 → 尿酸

关键酶：黄嘌呤氧化酶

」 嘌呤代谢异常会导致痛风，嘧啶代谢异常与肿瘤发生相关！

四大分子相互转化

糖类 ↔ 脂类

  • 糖类→脂类：乙酰CoA是共同中间产物，过量葡萄糖可转化为脂肪酸
  • 脂类→糖类：甘油可经糖异生转化为葡萄糖，但脂肪酸不能净生成糖

糖类 ↔ 蛋白质

  • 糖类→蛋白质：糖代谢中间产物可转化为非必需氨基酸的碳骨架
  • 蛋白质→糖类：生糖氨基酸可经糖异生途径转化为葡萄糖

脂类 ↔ 蛋白质

  • 脂类→蛋白质：甘油可转化为某些氨基酸，但脂肪酸转化有限
  • 蛋白质→脂类：生酮氨基酸可转化为乙酰CoA进而合成脂肪酸

核酸 ↔ 其他

  • 核酸→其他：核苷酸降解产物可进入TCA循环或糖异生
  • 其他→核酸：PRPP是关键中间物，氨基酸和糖提供前体
