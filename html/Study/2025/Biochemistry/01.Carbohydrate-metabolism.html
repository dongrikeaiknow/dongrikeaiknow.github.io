糖代谢概述 ✨

糖代谢是生物体获取能量的主要途径！成人每天所需能量的55%-60%来自糖代谢！

糖在体内以糖原的形式贮存，以葡萄糖的形式在血液中运输。1g葡萄糖在体内氧化可产生16.7 kJ的能量！

糖的分解代谢

大分子糖降解成单糖后，进一步氧化成CO₂和H₂O，释放能量！包括：

  • 糖酵解
  • 柠檬酸循环
  • 戊糖磷酸途径

糖的合成代谢

绿色植物利用日光、CO₂和水合成葡萄糖（光合作用）！动物则：

  • 合成糖原
  • 糖异生作用

1. 糖酵解途径 (Glycolysis) 🍬

葡萄糖 → 丙酮酸的过程！在细胞质中进行，无氧条件下也能进行！

关键特点

  • 10步酶促反应
  • 净生成2 ATP
  • 生成2 NADH

关键酶

己糖激酶 果糖磷酸激酶 丙酮酸激酶

能量变化

1 mol葡萄糖 → 2 mol丙酮酸

净生成：2 ATP

糖酵解详细步骤：

第一阶段：己糖磷酸酯的生成

1. 葡萄糖 → 6-磷酸葡糖

酶: 己糖激酶 (关键酶!)

ATP提供磷酸基团，消耗1 ATP

葡萄糖: HOCH₂(CHOH)₄CHO
→ 6-磷酸葡糖: HOCH₂(CHOH)₄CH₂OPO₃²⁻
ΔG°' = -16.7 kJ/mol

2. 6-磷酸葡糖 → 6-磷酸果糖

酶: 磷酸己糖异构酶

6-磷酸葡糖: HOCH₂(CHOH)₄CH₂OPO₃²⁻
⇌ 6-磷酸果糖: HOCH₂CO(CHOH)₃CH₂OPO₃²⁻
ΔG°' = +1.7 kJ/mol

3. 6-磷酸果糖 → 1,6-二磷酸果糖

酶: 磷酸果糖激酶 (关键酶!)

消耗1 ATP

6-磷酸果糖: HOCH₂CO(CHOH)₃CH₂OPO₃²⁻
→ 1,6-二磷酸果糖: HOCH₂CO(CHOH)₃CH₂OPO₃²⁻OPO₃²⁻
ΔG°' = -14.2 kJ/mol

第二阶段：丙糖磷酸的生成

4. 1,6-二磷酸果糖 → 磷酸二羟丙酮 + 3-磷酸甘油醛

酶: 醛缩酶

1,6-二磷酸果糖: HOCH₂CO(CHOH)₃CH₂OPO₃²⁻OPO₃²⁻
→ 磷酸二羟丙酮: CH₂OPO₃²⁻COCH₂OH
+ 3-磷酸甘油醛: CH₂OPO₃²⁻CHOHCHO
ΔG°' = +23.8 kJ/mol

5. 磷酸二羟丙酮 ⇌ 3-磷酸甘油醛

酶: 丙糖磷酸异构酶

磷酸二羟丙酮: CH₂OPO₃²⁻COCH₂OH
⇌ 3-磷酸甘油醛: CH₂OPO₃²⁻CHOHCHO

最终相当于1分子1,6-二磷酸果糖生成2分子3-磷酸甘油醛！

第三阶段：3-磷酸甘油醛 → 丙酮酸

6. 3-磷酸甘油醛 → 1,3-二磷酸甘油酸

酶: 3-磷酸甘油醛脱氢酶

生成1 NADH

3-磷酸甘油醛: CH₂OPO₃²⁻CHOHCHO
→ 1,3-二磷酸甘油酸: CH₂OPO₃²⁻CHOHCOOPO₃²⁻
ΔG°' = +6.3 kJ/mol

7. 1,3-二磷酸甘油酸 → 3-磷酸甘油酸

酶: 甘油酸磷酸激酶

生成1 ATP (底物水平磷酸化)

1,3-二磷酸甘油酸: CH₂OPO₃²⁻CHOHCOOPO₃²⁻
→ 3-磷酸甘油酸: CH₂OPO₃²⁻CHOHCOOH
ΔG°' = -18.5 kJ/mol

8. 3-磷酸甘油酸 → 2-磷酸甘油酸

酶: 磷酸甘油酸变位酶

3-磷酸甘油酸: CH₂OPO₃²⁻CHOHCOOH
→ 2-磷酸甘油酸: CH₂OHCHOPO₃²⁻COOH
ΔG°' = +4.6 kJ/mol

9. 2-磷酸甘油酸→ 磷酸烯醇丙酮酸

酶: 烯醇化酶

2-磷酸甘油酸: CH₂OHCHOPO₃²⁻COOH
→ 磷酸烯醇丙酮酸: CH₂=C(OPO₃²⁻)COOH
ΔG°' = -3.35 kJ/mol

10. 磷酸烯醇丙酮酸 → 丙酮酸

酶: 丙酮酸激酶 (关键酶!)

生成1 ATP (底物水平磷酸化)

磷酸烯醇丙酮酸: CH₂=C(OPO₃²⁻)COOH
→ 丙酮酸: CH₃COCOOH
ΔG°' = -31.4 kJ/mol

糖酵解总结 🎯

1 葡萄糖 → 2 丙酮酸

净生成: 2 ATP + 2 NADH

无氧条件下：丙酮酸 → 乳酸 (动物) 或乙醇 (酵母)

有氧条件下：丙酮酸进入线粒体继续氧化！

2. 丙酮酸氧化脱羧生成乙酰CoA 🔥

丙酮酸进入线粒体，在有氧条件下氧化脱羧生成乙酰CoA！这是连接糖酵解和三羧酸循环的关键步骤！

关键特点

  • 在线粒体基质中进行
  • 不可逆反应
  • 生成1 NADH

关键酶

丙酮酸脱氢酶系

能量变化

1 mol丙酮酸 → 1 mol乙酰CoA

生成: 1 NADH (≈2.5 ATP)

反应过程

丙酮酸 + CoA + NAD⁺ → 乙酰CoA + CO₂ + NADH + H⁺

酶: 丙酮酸脱氢酶系 (多酶复合体)

辅酶: TPP, 硫辛酸, CoA, FAD, NAD⁺

丙酮酸: CH₃-CO-COOH
→ 乙酰CoA: CH₃-CO~SCoA
ΔG°' = -33.4 kJ/mol

丙酮酸脱氢酶系组成 🧩

E₁: 丙酮酸脱氢酶

辅酶: TPP

E₂: 二氢硫辛酰转乙酰基酶

辅酶: 硫辛酸

E₃: 二氢硫辛酰脱氢酶

辅酶: FAD, NAD⁺

3. 柠檬酸循环 (三羧酸循环, TCA循环) 🍋

乙酰CoA彻底氧化为CO₂和H₂O的过程！在线粒体基质中进行，是三大营养物质分解的共同途径！

关键特点

  • 8步反应循环
  • 生成3 NADH, 1 FADH₂, 1 GTP
  • 两次脱羧生成2 CO₂

关键酶

柠檬酸合酶 异柠檬酸脱氢酶 α-酮戊二酸脱氢酶系

能量变化

1 mol乙酰CoA → 10 mol ATP

1 mol葡萄糖 → 25 mol ATP

柠檬酸循环详细步骤：

1. 乙酰CoA + 草酰乙酸 → 柠檬酸

酶: 柠檬酸合酶 (关键酶!)

乙酰CoA: CH₃-CO~SCoA
+ 草酰乙酸: -OOC-CH₂-CO-COO⁻
→ 柠檬酸: HOOC-CH₂-C(OH)(COOH)-CH₂-COOH
ΔG°' = -32.2 kJ/mol

2. 柠檬酸 ⇌ 顺乌头酸 ⇌ 异柠檬酸

酶: 顺乌头酸酶

柠檬酸: HOOC-CH₂-C(OH)(COOH)-CH₂-COOH
⇌ 顺乌头酸: HOOC-CH=C(COOH)-CH₂-COOH
⇌ 异柠檬酸: HOOC-CH₂-CH(COOH)-CH(OH)-COOH
ΔG°' = +13.3 kJ/mol

3. 异柠檬酸 → α-酮戊二酸 + CO₂

酶: 异柠檬酸脱氢酶 (关键酶!)

生成1 NADH

异柠檬酸: HOOC-CH₂-CH(COOH)-CH(OH)-COOH
→ α-酮戊二酸: HOOC-CH₂-CH₂-CO-COOH + CO₂
ΔG°' = -20.9 kJ/mol

4. α-酮戊二酸 → 琥珀酰CoA + CO₂

酶: α-酮戊二酸脱氢酶系 (关键酶!)

生成1 NADH

α-酮戊二酸: HOOC-CH₂-CH₂-CO-COOH
→ 琥珀酰CoA: HOOC-CH₂-CH₂-CO~SCoA + CO₂
ΔG°' = -33.5 kJ/mol

5. 琥珀酰CoA → 琥珀酸

酶: 琥珀酸硫激酶

底物水平磷酸化，生成1 GTP (≈ATP)

琥珀酰CoA: HOOC-CH₂-CH₂-CO~SCoA
→ 琥珀酸: HOOC-CH₂-CH₂-COOH
ΔG°' = -2.9 kJ/mol

6. 琥珀酸 → 延胡索酸

酶: 琥珀酸脱氢酶

生成1 FADH₂

琥珀酸: HOOC-CH₂-CH₂-COOH
→ 延胡索酸: HOOC-CH=CH-COOH
ΔG°' = 0 kJ/mol

7. 延胡索酸 → 苹果酸

酶: 延胡索酸酶

延胡索酸: HOOC-CH=CH-COOH
→ 苹果酸: HOOC-CH₂-CH(OH)-COOH
ΔG°' = -3.8 kJ/mol

8. 苹果酸 → 草酰乙酸

酶: 苹果酸脱氢酶

生成1 NADH

苹果酸: HOOC-CH₂-CH(OH)-COOH
→ 草酰乙酸: HOOC-CH₂-CO-COOH
ΔG°' = +29.7 kJ/mol

柠檬酸循环总结 🎯

1 乙酰CoA → 2 CO₂ + 3 NADH + 1 FADH₂ + 1 GTP

能量总计: 10 ATP/乙酰CoA

1 葡萄糖 → 2 乙酰CoA → 25 ATP

加上糖酵解和丙酮酸脱氢生成的ATP，1葡萄糖彻底氧化净生成32 ATP！

4. 戊糖磷酸途径 (PPP) 🌟

葡萄糖直接氧化脱羧的途径！主要生成NADPH和5-磷酸核糖！

关键特点

  • 在细胞质中进行
  • 生成NADPH
  • 提供5-磷酸核糖

关键酶

6-磷酸葡糖磷酸脱氢酶 6-磷酸葡糖酸脱氢酶

主要产物

NADPH (还原力)

5-磷酸核糖 (核酸合成)

氧化阶段 (6-磷酸葡糖 → 5-磷酸核糖)：

1. 6-磷酸葡糖 → 6-磷酸葡糖酸内酯

酶: 葡糖-6-磷酸脱氢酶

生成1 NADPH

6-磷酸葡糖: HOCH₂(CHOH)₄CH₂OPO₃²⁻
→ 6-磷酸葡糖酸内酯: O=CH(CHOH)₄CH₂OPO₃²⁻

2. 6-磷酸葡糖酸内酯 → 6-磷酸葡糖酸

酶: 内酯酶

6-磷酸葡糖酸内酯: O=CH(CHOH)₄CH₂OPO₃²⁻
→ 6-磷酸葡糖酸: HOOC(CHOH)₄CH₂OPO₃²⁻

3. 6-磷酸葡糖酸 → 5-磷酸核酮糖 + CO₂

酶: 6-磷酸葡糖酸脱氢酶

生成1 NADPH

6-磷酸葡糖酸: HOOC(CHOH)₄CH₂OPO₃²⁻
→ 5-磷酸核酮糖: CH₂OPO₃²⁻CO(CHOH)₂CH₂OH + CO₂

4. 5-磷酸核酮糖 ⇌ 5-磷酸核糖

酶: 磷酸戊糖异构酶

5-磷酸核酮糖: CH₂OPO₃²⁻CO(CHOH)₂CH₂OH
⇌ 5-磷酸核糖: CH₂OPO₃²⁻CHOH(CHOH)₃CHO

戊糖磷酸途径总结 🎯

1 6-磷酸葡糖 → 1 5-磷酸核糖+ 2 NADPH + 1 CO₂

生物学意义:

  • 提供NADPH (脂肪酸、固醇合成)
  • 提供5-磷酸核糖 (核酸合成)
  • 使不同碳数的糖相互转化

5. 乙醛酸循环 🌱

三羧酸循环的支路！存在于植物和微生物中，可将乙酰CoA转化为四碳化合物！

关键特点

  • 绕过TCA循环的脱羧步骤
  • 净合成四碳化合物
  • 油料种子萌发时重要

关键酶

异柠檬酸裂解酶 苹果酸合酶

生物学意义

脂肪 → 糖的转化

乙酸作为唯一碳源

乙醛酸循环反应：

1. 异柠檬酸 → 琥珀酸 + 乙醛酸

酶: 异柠檬酸裂解酶

异柠檬酸: HOOC-CH₂-CH(COOH)-CH(OH)-COOH
→ 琥珀酸: HOOC-CH₂-CH₂-COOH
+ 乙醛酸: O=CH-COOH

2. 乙醛酸 + 乙酰CoA → 苹果酸

酶: 苹果酸合酶

乙醛酸: O=CH-COOH
+ 乙酰CoA: CH₃-CO~SCoA
→ 苹果酸: HOOC-CH₂-CH(OH)-COOH

乙醛酸循环总结 🎯

2 乙酰CoA → 1 琥珀酸

生物学意义:

  • 油料种子萌发时，脂肪转化为糖
  • 微生物利用乙酸作为唯一碳源
  • 补充TCA循环中间物

6. 糖异生作用 ⏪

非糖物质（乳酸、甘油、氨基酸等）转变为葡萄糖或糖原的过程！主要在肝脏中进行！

关键特点

  • 基本上是糖酵解的逆过程
  • 消耗6高能磷酸键
  • 维持血糖恒定

关键酶

丙酮酸羧化酶 PEP羧化激酶 1,6-二磷酸果糖酶 6-磷酸葡糖酶

前体物质

乳酸、甘油

生糖氨基酸

TCA循环中间物

糖异生关键反应：

1. 丙酮酸 → 草酰乙酸 → 磷酸烯醇丙酮酸

酶: 丙酮酸羧化酶, PEP羧化激酶

消耗1 ATP + 1 GTP

丙酮酸: CH₃-CO-COOH
→ 草酰乙酸: -OOC-CH₂-CO-COO⁻
→ 磷酸烯醇丙酮酸: CH₂=C(OPO₃²⁻)COOH

2. 1,6-二磷酸果糖 → 6-磷酸果糖

酶: 果糖-1,6-二磷酸酶

1,6-二磷酸果糖: HOCH₂CO(CHOH)₃CH₂OPO₃²⁻OPO₃²⁻
→ 6-磷酸果糖: HOCH₂CO(CHOH)₃CH₂OPO₃²⁻

3. 6-磷酸葡糖 → 葡萄糖

酶: 葡糖-6-磷酸酶

6-磷酸葡糖: HOCH₂(CHOH)₄CH₂OPO₃²⁻
→ 葡萄糖: HOCH₂(CHOH)₄CHO

乳酸循环 (Cori循环)

肌肉中糖酵解产生乳酸 → 血液 → 肝脏中糖异生为葡萄糖 → 血液 → 肌肉

2 乳酸 + 6 ATP → 1 葡萄糖

糖异生总结 🎯

2 丙酮酸 → 1 葡萄糖

消耗: 4 ATP + 2 GTP

生理意义:

  • 维持血糖恒定
  • 清除乳酸
  • 氨基酸代谢途径

糖代谢全景图 🌈

糖代谢各途径相互联系，构成复杂的代谢网络！以下是主要代谢途径的关系：

糖代谢途径关系

多糖 → 单糖 → 糖酵解 → 丙酮酸

↓ (无氧) ↓ (有氧)

乳酸/乙醇 乙酰CoA → TCA循环

↑ (糖异生) ↖ (乙醛酸循环)

非糖物质 ← 戊糖磷酸途径

能量总结

  • 糖酵解: 1葡萄糖 → 2 ATP + 2 NADH
  • 丙酮酸→乙酰CoA: 1 NADH
  • TCA循环: 1乙酰CoA → 10 ATP
  • 总计: 1葡萄糖 → 32 ATP
