List24 词汇卡片

前缀：re– 表示"一再"或"向后"

renew v. 更新；续借；重新开始

词根词缀：re-(再) + new(新的) → 再次变新 → 更新

例句：I need to renew my passport before my trip.

我需要在旅行前更新我的护照。

记忆技巧：

"re"表示"再次"，"new"是"新的"，合起来就是"再次变新"！

🔁

resolve v. 解决；决定 n. 决心

词根词缀：re-(再) + solve(解决) → 再次解决 → 下定决心

例句：They met to resolve the dispute between them.

他们会面以解决彼此间的争议。

记忆技巧：

"solve"是"解决"，"re"表示"反复"，反复思考解决方案就是"下定决心"！

💪

相近词汇总结

1. 记录相关

  • record (v. 记录 n. 记录) - re-(再) + cord(心) → 再次记在心里
  • register (v. 登记 n. 登记簿) - re-(回) + gister(带来) → 带回来记录
  • recall (v. 回忆 n. 记忆力) - re-(再) + call(叫) → 再次叫回记忆

记忆技巧：

这些词都与"记录"或"记忆"有关："record"是记录在心，"register"是登记在册，"recall"是回忆往事！

2. 恢复相关

  • recover (v. 恢复) - re-(再) + cover(覆盖) → 再次覆盖恢复
  • restore (v. 恢复；修复) - re-(再) + store(储存) → 再次储存原状
  • renew (v. 更新；重新开始) - re-(再) + new(新的) → 再次变新

记忆技巧：

这些词都表示"恢复"或"更新"："recover"侧重恢复健康，"restore"侧重修复原状，"renew"侧重更新状态！

3. 回应相关

  • response (n. 响应；回答) - re-(回) + sponse(承诺) → 回应承诺
  • responsible (adj. 负责任的) - re-(回) + spons(承诺) + -ible → 能回应承诺的
  • recommend (v. 推荐) - re-(加强) + commend(称赞) → 强烈称赞推荐

记忆技巧：

这些词都与"回应"或"承诺"有关："response"是回应，"responsible"是能回应的(负责任的)，"recommend"是强烈回应(推荐)！

4. 重复动作

  • refine (v. 精炼) - re-(再) + fine(精细的) → 再次变得精细
  • resolve (v. 解决；决定) - re-(再) + solve(解决) → 再次解决
  • replace (v. 替换) - re-(再) + place(放置) → 再次放置

记忆技巧：

这些词都表示"重复动作"："refine"是反复变精细，"resolve"是反复解决，"replace"是再次放置新的！
